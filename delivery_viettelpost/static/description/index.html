Delivery Viettelpost

The Delivery Viettelpost addon is an Odoo integration that connects seamlessly to ViettelPost's API, enabling efficient domestic and international package delivery and courier management. It streamlines shipping logistics, tracking, and improves overall delivery processes within the Odoo ERP system for a smooth customer experience.

Hot Features

Create/Update/Cancel Order Waybill
Automatically create Purchase Orders for shipping costs
Estimated fee delivery
Track order waybill with webhook
Print order waybill
Tracking request API
Tracking request API from webhook
Register for a warehouse to send goods
Resource sync: The post offices list, The address list, The status list,... in one click
Support language: Vietnamese & English
